# Игра "Сапер"

Веб-версия классической игры "Сапер", реализованная на JavaScript с использованием IndexedDB для хранения данных.

## Описание игры

Сапер - это логическая компьютерная игра, где игрок должен открыть все ячейки на поле, не попав на мину. При открытии ячейки показывается число, указывающее количество мин в соседних ячейках.

## Функциональность

- **Новая игра**: Начало новой игры с настраиваемыми параметрами
- **Список партий**: Просмотр всех сохраненных игр
- **Повтор партии**: Воспроизведение ходов из любой сохраненной игры

## Как играть

1. Нажмите "Новая игра" в главном меню
2. Введите имя игрока, размер поля и количество мин
3. Нажмите "Начать игру"
4. Кликните левой кнопкой мыши по ячейке, чтобы открыть ее
5. Кликните правой кнопкой мыши, чтобы поставить/убрать флажок
6. Откройте все ячейки без мин, чтобы выиграть

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- IndexedDB для хранения данных

## Запуск игры

Откройте файл `index.html` в веб-браузере или перейдите по ссылке GitHub Pages:

- `index.html` - главный HTML файл
